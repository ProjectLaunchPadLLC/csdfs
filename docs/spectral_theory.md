# Spectral Theory

## Definition

Let Σ(t) be the covariance matrix of system states.

Eigen decomposition:

Σ(t) = V Λ V⁻¹

---

## Metastable Eigenmodes

A mode ν is stable if:

- Var(λ_i) < ε
- P(λ_i > threshold) > 0.8

---

## Interpretation

- High eigenvalue → dominant cognitive direction
- Stability → persistence of structure
- Drift → phase transition signal
